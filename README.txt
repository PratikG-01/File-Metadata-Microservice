FILE METADATA MICROSERVICE

OVERVIEW
This microservice allows users to upload a file and returns metadata about the file including its 
name, type, and size. It is built using Node.js, Express, and Multer for handling file uploads.

FEATURES

    •    Accepts file uploads via POST request

    •    Extracts and returns file name, type, and size

    •    Returns results in JSON format

PREREQUISITES
    •    Node.js and npm installed

INSTALLATION
    •    Clone the repository

    •    Navigate into the project folder

    •    Run npm install to install dependencies

    •    Start the server using npm start or node index.js

TECHNOLOGIES USED
    •    Node.js
    •    Express.js
    •    Multer (middleware for handling multipart/form-data)

PROJECT STRUCTURE
    •    index.js – Main application file
    •    package.json – Project dependencies and scripts
